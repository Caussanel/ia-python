def id3.Noeud.estTerminal (   self  ) 

Indique si le noeud est terminal ou non.

Cette méthode indique si le noeud considéré est terminal ou pas. Un noeud est terminal ssi il ne possède pas d'attribut de partitionnement, mais uniquement des exemples classifiés.

Paramètres:
[in] self référence automatique vers l'objet exécutant cette méthode
Renvoie:
True si le noeud est terminal, False sinon

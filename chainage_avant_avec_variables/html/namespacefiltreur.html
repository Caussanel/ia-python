def filtreur.filtrer (   datum,
  pattern 
)

Effectue un filtrage entre deux propositions

Le filtrage est effectue en parcourant les deux propositions en meme temps, en prenant deux elements, en les comparant et en construisant les substitutions qui s'imposent. Par exemple:

  • prop1 = [ 'Superman', 'a', 'une', 'cape', 'rouge' ]
  • prop2 = [ '?x', 'a', 'une', '?y', 'rouge' ]

produit { '?x':Superman, '?y':'cape' }

L'algorithme est le suivant:

  1. function FILTRER( datum, pattern )
  2.   if pattern est un symbol:
  3.       if datum et pattern sont identiques: return {}
  4.       if pattern est une variable: return {pattern : datum}
  5.       return ECHEC
  6.   if datum est un symbol: return ECHEC
  7.   F1 <- premier element de datum, T1 <- reste de datum
  8.   F2 <- premier element de pattern, T2 <- reste de pattern
  9.   Z1 <- FILTRER(F1,F2)
 10.   if Z1 = ECHEC: return ECHEC
 11.   G1 <- T1
 12.   G2 <- remplacer les variables de T2 par les unifications Z1
 13.   Z2 <- FILTRER(G1,G2)
 14.   if Z2 = ECHEC: return ECHEC
 15.   return { Z1 union Z2 }

Paramètres:
datum la propopsition sans variable
pattern la proposition avec variables
Renvoie:
les substitutions si le filtrage a reussi: {'?x':'a',..}, 'ECHEC' sinon

Effectue un filtrage en tenant compte d'une liste de substitutions initiale

Cette fonction effectue un filtrage sur deux propositions. Cette fonction peut prendre en argument un environnement initial. Un environnement initial est une liste de substitutions. Le resultat final est l'union de l'environnement initial avec les substitutions donnees par le filtrage

Paramètres:
pattern une proposition pouvant contenir des variables
datum la proposition sans variables
environnement une liste initiale de substitutions
Renvoie:
filtre(proposition,datum) si OK, ECHEC sinon

def filtreur.substitueVariables (   pattern,
  substitutions 
)

Retourne la proposition avec les variables remplacees par leurs valeurs

Cette methode se charge de substituer toutes les variables de la proposition passee en argument par leur valeur associee dans la liste des substitutions. Si la proposition est une variable, alors il suffit de retourner la valeur associee si elle existe, sinon, on teste chaque element de la proposition recursivement

Paramètres:
[in] pattern la proposition (contient des variables, atomes,... sous forme [' ',' ', ' '] OU ' ' ou ['[]'...])
[in] substitutions la liste des substitutions {a:b, c:d,...}
Renvoie:
la proposition avec les variables remplacees par leur valeurs

def filtreur.unionSubstitutions (   substitution1,
  substitution2 
)

Construit l'union de deux substitutions

Pour construire l'union de deux substitutions, on prend chaque variable de la liste substitution2. Si la variable existe deja dans substitution1, alors sa valeur est mise a jour, sinon elle est creee avec sa valeur associee.

Paramètres:
[in] substitution1 la premiere substitution
[in] substitution2 la deuxieme substitution
Renvoie:
substitution1 UNION substitution2; echec if substitution1 ou substitution2 ne sont pas des substitutions

Description détaillée

Classe gerant la liste des noeuds NOEUDS.

Cette classe est une sorte de wrapper pour l'object NOEUDS facilitant la manipulation de celui-ci. Les fonctions disponibles peuvent ajouter ou obtenir un noeud donne. D'autres fonctions avancees permettent d'appliquer la consistance des noeuds a tous les noeuds ou d'initialiser une instance des variables presentes.

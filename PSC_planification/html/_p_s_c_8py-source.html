00365 ## Algorithme de backtracking simple
00366 #
00367 # Le principe de l'algorithme est de tenter d'assigner une valeur a chaque variable
00368 # dans la liste PSC.VAR. A chaque assignation, on verifie que toutes le contraintes
00369 # liant la variable courante avec les variables deja instanciees sont satisfaites. 
00370 # Sinon, on procede a un backtrack pour essayer une autre valeur. 
00371 #
00372 # @param[in] k la profondeur de la recherce (commence a 0)
00373 # @param[in] toutesLesSolutions si \b True alors cherche TOUTES les solutions,
00374 #                       sinon s'arrete a la premiere (valeur par defaut)
00375 # @param[in] initialise initialise les variables globales ITERATIONS et SOLUTIONS

00381         algo="bt"
00382

00384         global ITERATIONS
00385         global SOLUTIONS
00386 
00387         # faut-il ajouter toutes les solutions ?
00388         if toutesLesSolutions==None:
00389                 toutesLesSolutions=False
00390 
00391         # faut-il initialiser les variables ?
00392         if initialise==None or initialise==True:
00393                 ITERATIONS=0
00394                 SOLUTIONS=[]
00395                 libPSC.NBCONTRAINTES=0
00396                 initialise=False

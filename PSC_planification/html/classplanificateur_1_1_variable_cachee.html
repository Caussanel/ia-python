Description détaillée

Cette classe represente une variable cachee entre deux variables de proposition et une variable d'action.

Avertissement:
Le domaine est defini comme ceci : self.domaine=[valeur_1,....,valeur_i] ou valeur_i:= [bool1, op, bool2] bool1 : la valeur pour la precondition (variable de proposition) op : la valeur pour l'operateur (variable d'action) bool2 : la valeur pour la postcondition (variable de proposition)

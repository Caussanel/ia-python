00145         ## Constructeur de la classe Noeud
00146         #
00147         # Initialise l'etat interne d'un noeud de recherche. Il comprend
00148         # notamment une reference vers un noeud du graphe, ainsi qu'un cout
00149         # f(n) et un cout c(n). La relation entre ces deux elements est la

00192         ## Teste si le noeud passe en parametre est solution

00208         ## Met a jour le cout \f$c_{n}\f$ du noeud passe en parametre
00209         #
00210         # Attention, la mise a jour s'effectue bien sur le noeud passe en 
00211         # parametre, et non pas le noeud represente par l'objet traitant
00212         # cette methode. Cela est tout a fait normal. Pour chaque noeud, il y
00213         # a 0,1, ou plusieurs successeurs. L'algorithme de recherche part du
00214         # premier noeud et descend selon les successeurs. Alors, pour le noeud
00215         # courant on a toujours un cout defini(0 pour le noeud initial). A chaque
00216         # successeur du noeud, on met a jour a partir du cout du noeud courant et
00217         # de la distance euclidienne les separant :
00218         #
00219         # \f[c_{succ} = c_{self} + \delta (succ,self)\f]
00220         # 
00221         # @param[in] self un pointeur vers cet objet lui-meme (automatique)
00222         # @param[in,out] succ le noeud a mettre a jour
00223         def metAJourCoutC( self, succ ):
00224                 succ.coutC = self.coutC + self.refElement.distanceEuclidienne(
00225                                                                                                                         succ.refElement )

00228         ## Met a jour le cout \f$f_{n}=c_{n}+h_{n}\f$ du noeud passe en parametre
00229         #
00230         # Met a jour le cout total du noeud courant. Pour mettre a jour cette
00231         # valeur, il faut que le cout c(n) soit a jour. La partie heuristique
00232         # h(n) de la formule est donnee par la distance euclidienne entre le
00233         # noeud self et le noeud but passe en parametre.
00234         # Dans notre cas, self est le noeud but !!!!
00235         #
00236         # \f[f_{succ} = c_{succ} + \delta (succ,self)\f]
00237         # 
00238         # @param[in] self un pointeur vers cet objet lui-meme (automatique)
00239         # @param[in,out] succ le noeud a mettre a jour
00240         # @note h(n)=distance euclidienne entre self et le but
00241         def metAJourCoutF(self,succ):
00242                 succ.coutF = succ.coutC + succ.refElement.distanceEuclidienne(
00243                                                                                                                         self.refElement )

00254

00257                 succs = map( Noeud, self.refElement.voisins )

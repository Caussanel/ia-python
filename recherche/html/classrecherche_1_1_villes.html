Description détaillée

Representation d'un graphe de villes.

Le graphe est represente par les relations de voisinage entre les villes. Toutes les villes sont ajoutees dans le dictionnaire CARTE

Cherche et retourne une ville associee a un nom donne.

Cette methode se contente de retourner le noeud associe au nom passe en parametre, si ce nom est deja dans le dictionnaire. Si ce n'est pas le cas, alors chercheVille() leve une exception du type VILLE_PAS_TROUVEE. Cette exception contient le nom de la ville qui n'a pas ete trouvee

Paramètres:
[in] self un pointeur vers cet objet lui-meme (automatique)
[in] nom le nom de la ville
Renvoie:
Un pointeur vers le noeud recherche s'il existe, -1 sinon
Exceptions:
Retourne une exception du type VILLE_PAS_TROUVEE pour signifier que la ville en question n'a pas ete trouvee

Definit une route (un arc) entre deux villes.

Un arc dans un graphe est defini par une relation de voisinage. On determine s'il existe une route entre deux villes, si chacune est presente dans le liste des voisins de l'autre. Donc, pour creer une route entre les deux, il suffit de les ajouter en tant que voisins

Paramètres:
[in] self un pointeur vers cet objet lui-meme (automatique)
[in] nom_ville1 le nom de la premiere ville
[in] nom_ville2 le nom de la deuxieme ville

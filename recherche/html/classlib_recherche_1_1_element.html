Description détaillée

Represente un noeud d'un graphe.

La classe Element correspond a un simple noeud d'un graphe. Dans ce cas il s'agira d'un graphe de villes. Donc un element sera une ville.

Verifie si le nom passe en param est egal a celui de cet element.

Cette methode compare deux chaines de caracteres et teste si elles sont egales. Si elles sont egales, alors le nom passe en parametre est le meme que celui de cet element.

Paramètres:
[in] self un pointeur vers cet objet lui-meme (automatique)
[in] un_nom le nom avec lequel il faut comparer
Renvoie:
True si les deux noms correspondent, False sinon

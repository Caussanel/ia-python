Description détaillée

Classe modelisant une contrainte binaire.

Une contrainte binaire est representee dans un graphe par un arc allant d'un noeud source(noeud de depart) a un noeud de destination(noeud d'arrivee). Cette contrainte est souvent de la forme v1<v2, v1==v2, v1>v2, etc...

def libPSC.ContrainteBinaire.reviser (   self  ) 

Applique l'algorithme REVISER.

Pour chaque variable, pour chaque valeur v dans le domaine de cette variable, verifie s'il existe une valeur possible pour la deuxieme variable qui satisfasse la contrainte ; sinon, supprime v du domaine.

Paramètres:
[in] self reference automatique vers l'instance executant cette methode.
Note:
Regarder le chapitre 8.5.2 du cours pour de plus amples informations sur le fonctionnement de l'algorithme de Waltz et REVISER

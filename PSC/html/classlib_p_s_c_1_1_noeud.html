Description détaillée

Classe modelisant un Noeud.

Un noeud est un element d'un graphe et fait reference a une variable. Un noeud partage certaines contraintes avec lui-meme ou avec d'autres noeuds. Ces contraintes sont exprimees a l'aide d'arcs. Un arc partant d'un noeud a lui-meme exprime une contrainte unaire. Un arc partant d'un noeud a un autre exprime une contrainte binaire.

def libPSC.Noeud.consistanceDuNoeud (   self,
  contraintes 
)

Applique la consistance du noeud sur ce noeud.

La methode de consistance consiste a supprimer du domaine toutes les valeurs qui violent les contraintes. Pour cela, il suffit d'iterer parmi toutes les valeurs possibles du domaine et de verifier a chaque fois si la variable respecte bien toutes les contraintes. Dans la negative, il suffit de supprimer la valeur du domaine.

Paramètres:
[in] self reference automatique vers l'instance executant cette methode.
[in] contraintes la liste de contraintes unaires pour ce noeud
Voir également:
Chapitre 8.5.1 du livre
Avertissement:
lors de l'iteration des valeurs du domaine, il faut bien faire attention a iterer sur une copie du domaine, car on ne peut pas supprimer des valeurs d'une liste et continuer a iterer sur celle-ci
